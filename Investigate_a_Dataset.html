Introduction¶

I will be analyzing the movie database (IMDb) to review correlations within the movie industry. A few questions I would like to answer with this analysis would be:

  1. Which decade spent/made the most money?
  2. Which prodcution companies make the most movies?
  3. Which production companies have the highest profits?
  4. Do you really need to spend money to make money?
  5. Which month is best to release a movie?
